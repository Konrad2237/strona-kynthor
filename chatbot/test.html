KYNTHOR Chatbot — strona testowa

Widget powinien pojawić się w prawym dolnym rogu.

Przed testem sprawdź:

  • Klucz ANTHROPIC_API_KEY ustawiony w Netlify: Site settings > Environment variables
  • Plik knowledge.txt obecny w folderze chatbot/
  • Zainstalowane zależności: npm install w folderze chatbot/
  • Deploy na Netlify wykonany (lub netlify dev uruchomiony lokalnie)
Widget się nie pojawia? Otwórz konsolę przeglądarki klawiszem F12 (lub Ctrl+Shift+I / Cmd+Option+I na Mac), przejdź do zakładki Console i sprawdź czy nie ma błędów ładowania plików widget.css lub widget.js.
